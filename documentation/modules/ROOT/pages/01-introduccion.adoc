== OpenScap

El NIST ha publicado un conjunto de especificaciones para estandarizar la gestión de información de seguridad, este conjunto de especificaciones recibe el nombre de **Security Content Automation Protocol** o **SCAP**.

**OpenSCAP** es una implementación Open Source de estas especificaciones. 

**OpenScap** provee un conjunto de utilidades que nos permiten comprobar el nivel de compliance de un conjunto de equipos así como generar playbooks de ansible o scripts bash para proceder a remediar todos aquellos puntos con los que no se sea compliance.
